# LAB: MySQL Database Creation, Population, and-Advanced-Querying

<h2>Description</h2>
  <p>This project demonstrates the design, implementation, and management of a relational database using MySQL. The database was built from an Entity Relationship Diagram (ERD) and includes the creation of multiple tables, primary and foreign key constraints, and relationships between entities. After constructing the database schema, sample data was imported and validated to ensure data integrity and proper table relationships.
<br />

  <p>The project also involves developing and executing a variety of SQL queries to retrieve, analyze, and summarize data. These queries include table joins, aggregate functions, filtering, grouping, subqueries, and views to generate meaningful insights from the database. Through these tasks, the project demonstrates fundamental database administration, data modeling, and SQL querying skills commonly used in business and enterprise database environments.
<br />
